# Mecard
Mecard是一个用python写成的简单的命令行工具，使用了fire框架，可以实现类似记忆卡片(memory card)的功能。他需要手动导入题库，然后抽查你对与题库中题目的理解程度。

## Getting Started
```
# start memory
python main.py start
```
